# ComprasUSA
> Aplicativo para desenvolvimento Swift/iOS

Aplicativo para calcular a conversão do cambio Dólar/Real e taxações.
Swift 5.4

## Conceitos abordados

* Auto Layout
    * Constraint
    * Autoresizing 
    * Size Classes

* UI Components
    * UIView
    * UIImage
    * UITabBarController
    * UIStackView
    * UISwitch
