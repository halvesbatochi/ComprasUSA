# ComprasUSA
> Aplicativo para desenvolvimento Swift/iOS

Aplicativo para calcular a conversão do câmbio Dólar/Real e taxações.


## Conceitos abordados

* Linguagem
    * Swift 5.4

* Auto Layout
    * Constraint
    * Autoresizing 
    * Size Classes
        * Modos de exibição Landscape e Portrait diferente

* UI Components
    * UIView
    * UIImage
    * UILabel
    * UITextField
    * UITabBarController
    * UIStackView
    * UISwitch

* Singleton
    * Variáveis
    * Variáveis computadas
    * Funções
    * NumberFormatter
    * Extension

* Delegates
    * UITextFieldDelegate

* Assets diferentes por device
